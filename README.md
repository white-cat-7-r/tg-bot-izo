# Telegram-бот для генерации изображений

Бот, который создаёт картинки по текстовому описанию. Бесплатно, без регистрации на сторонних сервисах.

## Что умеет

- Написать текстовое описание и выбрать стиль (аниме, мультфильм, киберпанк и др.)
- Сгенерировать уникальное изображение по описанию
- Использовать бесплатные токены ежедневно (50 токенов = 10 генераций)
- Покупать тарифы с увеличенным лимитом
- Смотреть историю генераций

## Что нужно для запуска

| Параметр | Где взять |
|----------|-----------|
| **Telegram Bot Token** | [@BotFather](https://t.me/BotFather) → `/newbot` |

Всё! Другие ключи не нужны — генерация картинок работает бесплатно.

## Пошаговая установка

### Шаг 1: Установи Python

1. Зайди на https://python.org/downloads
2. Нажми жёлтую кнопку "Download Python 3.x.x"
3. Запусти скачанный файл
4. **ОБЯЗАТЕЛЬНО** поставь галочку **"Add python to PATH"** (самая нижняя галочка на первом экране установки)
5. Нажми "Install Now"
6. Дождись надписи "Setup was successful"

### Шаг 2: Скачай проект

1. Зайди на страницу проекта на GitHub
2. Нажми зелёную кнопку "<> Code"
3. Нажми "Download ZIP"
4. Распакуй архив в любую папку, например на Рабочий стол

### Шаг 3: Открой чёрное окно (cmd) в папке проекта

1. Открой распакованную папку проекта (где лежит файл `bot.py`)
2. В верхней части окна проводника щёлкни на **строку адреса** (там где написан путь типа `C:\Users\...\tg-bot-izo`)
3. Строка адреса станет синей — выдели весь текст и **удали**
4. Напиши `cmd` и нажми **Enter**
5. Откроется чёрное окно — это консоль. **Не закрывай её!**

### Шаг 4: Установи библиотеки

В чёрном окне напиши ровно эту команду (можно скопировать, вставить правой кнопкой мыши):

```
pip install -r requirements.txt
```

Нажми **Enter**. Жди 1-2 минуты. Увидишь много текста — это нормально. В конце должно быть что-то вроде "Successfully installed ...".

**Если вместо этого видишь `>>>` (три угловые скобки)** — значит ты открыл не консоль (cmd), а сам Python. Закрой это окно и сделай Шаг 3 заново.

### Шаг 5: Настрой бота

1. В папке проекта найди файл `.env.example`
2. Нажми на него правой кнопкой мыши → **Переименовать**
3. Замени `.env.example` на `.env` (просто убери слово `.example`)
4. Нажми правой кнопкой мыши на файл `.env` → **Открыть с помощью** → **Блокнот**
5. Замени `YOUR_TELEGRAM_BOT_TOKEN` на твой токен бота
6. Сохрани файл: нажми **Ctrl+S** и закрой Блокнот

### Шаг 6: Запусти бота

В том же чёрном окне (консоли) напиши:

```
python bot.py
```

Нажми **Enter**. Должно появиться "Bot starting...". Консоль **не закроется** — бот работает.

### Шаг 7: Проверь

1. Открой Telegram
2. Найди своего бота по имени
3. Напиши `/start`
4. Нажми «Создать картинку»
5. Напиши что хочешь увидеть и выбери стиль

## Если что-то не работает

| Проблема | Решение |
|----------|---------|
| Консоль закрывается сразу | Ты запустил не из папки проекта. Сделай Шаг 3 заново |
| `>>>` вместо обычной строки | Ты открыл Python, а не cmd. Закрой, сделай Шаг 3 |
| `pip is not recognized` | Python установился без галочки "Add to PATH". Переустанови с галочкой |
| `No module named aiogram` | Не установил библиотеки. Сделай Шаг 4 |
| Бот не отвечает в Telegram | Проверь токен в файле `.env` |
| Ошибка при генерации картинки | Проверь интернет |

## Как добавить новый стиль

Открой `utils/styles.py` и добавь элемент в `STYLES`:

```python
"new_style": {
    "name": "Новый стиль",
    "description": "Описание стиля",
    "prompt_template": "new style, artistic, {prompt}",
},
```

```
handlers/           <- Команды бота
keyboards/          <- Кнопки в Telegram
utils/styles.py     <- Список стилей
```
