# Telegram-бот для генерации изображений

Бот, который создаёт картинки по текстовому описанию. Бесплатно, без регистрации на сторонних сервисах.

## Что умеет

- Написать текстовое описание и выбрать стиль (аниме, мультфильм, киберпанк и др.)
- Сгенерировать уникальное изображение по описанию
- Использовать бесплатные токены ежедневно (50 токенов = 10 генераций)
- Покупать тарифы с увеличенным лимитом
- Смотреть историю генераций

## Система токенов

| План | Токенов/день | Генераций | Цена |
|------|-------------|-----------|------|
| Бесплатный | 50 | 10 | 0₽ |
| Стандарт | 200 | 40 | 500₽/мес |
| Расширенный | 500 | 100 | 900₽/мес |
| Безлимит | ∞ | ∞ | 1300₽/мес |

- 1 генерация = 5 токенов
- Токены сбрасываются каждый день в полночь

## Что нужно для запуска

| Параметр | Где взять | Зачем |
|----------|-----------|-------|
| **Telegram Bot Token** | [@BotFather](https://t.me/BotFather) → `/newbot` | Идентификатор бота |

Всё! Другие ключи не нужны — генерация картинок работает бесплатно через Pollinations.ai.

## Установка

### 1. Установи Python 3.10+

Скачай с [python.org](https://python.org/downloads). При установке поставь галочку "Add Python to PATH".

### 2. Скачай проект

Нажми кнопку "Code" → "Download ZIP" на странице проекта. Распакуй архив в любую удобную папку.

### 3. Установи необходимые библиотеки

Открой папку проекта. В строке адреса проводника напиши `cmd` и нажми Enter — откроется чёрное окно.

В этом окне напиши (можно скопировать и вставить правой кнопкой мыши):

```
pip install -r requirements.txt
```

Нажми Enter. Дождись пока напишет "Successfully installed". Это займёт 1-2 минуты.

### 4. Настрой бота

В папке проекта найди файл `.env.example`. Нажми правой кнопкой мыши → "Переименовать" → замени на `.env`.

Открой файл `.env` правой кнопкой мыши → "Открыть с помощью" → "Блокнот". Заполни так:

```
BOT_TOKEN=сюда_вставь_токен_бота
DATABASE_URL=sqlite+aiosqlite:///./bot.db
TEST_USER_ID=0
TEST_MODE=true
```

Сохрани файл (Ctrl+S).

### 5. Запусти бота

В папке проекта в строке адреса проводника напиши `cmd` и нажми Enter. В открывшемся окне напиши:

```
python bot.py
```

Если всё правильно — увидишь "Bot starting...". Консоль не закроется.

### 6. Проверь

1. Открой Telegram
2. Найди своего бота по имени
3. Напиши `/start`
4. Нажми «Создать картинку»
5. Напиши описание и выбери стиль

## Тестовый режим

Для тестирования без реальной генерации:

1. В `.env` установи:
```
TEST_MODE=true
TEST_USER_ID=твой_telegram_id
```

2. Запусти: `python bot.py`

В тестовом режиме бот возвращает заглушку вместо реальной картинки.

## Как добавить новый стиль

Открой `utils/styles.py` и добавь элемент в `STYLES`:

```python
"new_style": {
    "name": "Новый стиль",
    "description": "Описание стиля",
    "prompt_template": "new style, artistic, {prompt}",
},
```

## Структура проекта

```
bot.py              <- Точка входа
config.py           <- Настройки из .env
database.py         <- Подключение к БД (SQLite)
models.py           <- Модели (пользователи, история)
ai_processor.py     <- Генерация изображений (Pollinations.ai)
token_manager.py    <- Логика токенов
handlers/           <- Хендлеры бота
  start.py          <- /start, главное меню
  process.py        <- Генерация изображений
  payment.py        <- Баланс и тарифы
  history.py        <- История генераций
keyboards/          <- Кнопки в Telegram
utils/styles.py     <- Список стилей
```

## Частые вопросы

### Бот не отвечает на /start
- Убедись, что `BOT_TOKEN` правильный
- Проверь логи в консоли

### Ошибка при генерации
- Попробуй другой стиль
- Проверь подключение к интернету

### Не хватает токенов
- Дождись сброса лимита (полночь)
- Купи тариф с большим лимитом
- Используй `TEST_MODE=true` для тестирования
